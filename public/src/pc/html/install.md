# 安装
#### 一键安装，命令行模式进入源码根目录
```
linux: 命令行模式下  
1、 ./ly install         #
2、 sudo ./ly install2   # 全局安装包
```
```
windows: 请先安装git，然后在git 命令行下输入
1、 ./ly install
2、 ./ly install2
```
```
mac: 在命令行下
1、  ./ly install
2、  ./ly install2
```

#### 手动安装方式，命令行模式进入源码根目录
1. `npm install gulp -g`
2. `npm install node-gyp -g`
3. `npm install nodemon -g`
4. `npm install`
5. `cd public && npm install`


#### sass安装需要额外安装以下npm包
`npm install gulp-sass`  
`npm install sass-loader`
